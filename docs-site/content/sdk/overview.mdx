# tinytsdk — Overview

**tinytsdk** is the TypeScript/JavaScript SDK for TinyTrack. It provides a typed WebSocket
client and a set of React components for displaying live system metrics.

## Packages

| Package | Size (gzip) | Description |
|---------|-------------|-------------|
| `tinytsdk` | 2.8 KB core / 23.7 KB with React | Full SDK: client + React components |
| `tinytsdk-lite` | 2.7 KB | Core only: client + protocol, no React |

## Install

```bash
npm install tinytsdk
```

## Quick start

```tsx
import { TinyTrackProvider, MetricsBar } from 'tinytsdk/react';

export default function App() {
  return (
    <TinyTrackProvider url="ws://your-host:25015">
      <MetricsBar />
    </TinyTrackProvider>
  );
}
```

## createClient()

```ts
import { createClient } from 'tinytsdk';

const client = createClient({
  url: 'ws://your-host:25015',
  password: 'optional-token',
  socket: {
    connectTimeout: 5000,
    reconnectStrategy: (retries) => Math.min(retries * 500, 5000),
  },
});

client.on('metrics', (m) => {
  console.log(`CPU: ${(m.cpu / 100).toFixed(1)}%  RAM: ${(m.mem / 100).toFixed(1)}%`);
});

await client.connect();
```

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `connect` | — | TCP connection established |
| `ready` | — | WS handshake done, streaming active |
| `metrics` | `TtMetrics` | New metrics frame |
| `reconnecting` | `{ attempt: number }` | Reconnect attempt |
| `disconnect` | — | Connection closed |
