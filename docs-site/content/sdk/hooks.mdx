# Hooks

All hooks must be used inside `TinyTrackProvider`.

## useMetrics()

Returns the latest metrics frame. Re-renders on every push (default: 1 s).

```ts
import { useMetrics } from 'tinytsdk/react';

function CpuBadge() {
  const m = useMetrics();
  if (!m) return <span>—</span>;
  return <span>{(m.cpu / 100).toFixed(1)}%</span>;
}
```

## useSysInfo()

Returns system info received during the WebSocket handshake.

```ts
import { useSysInfo } from 'tinytsdk/react';

function HostBadge() {
  const info = useSysInfo();
  return <span>{info?.hostname ?? '—'}</span>;
}
```

`TtSysInfo` fields:

| Field | Type | Description |
|-------|------|-------------|
| `hostname` | `string` | Server hostname |
| `os_type` | `string` | OS type (e.g. `Linux`) |
| `uptime_sec` | `number` | Uptime in seconds |
| `interval_ms` | `number` | Current push interval |
| `l1_capacity` | `number` | L1 ring buffer capacity |
| `l2_capacity` | `number` | L2 ring buffer capacity |
| `l3_capacity` | `number` | L3 ring buffer capacity |

## useHistory()

Requests and returns history from a ring buffer level.

```ts
import { useHistory } from 'tinytsdk/react';

function RecentCpu() {
  const frames = useHistory('l1', 60); // last 60 L1 entries
  return <span>avg: {average(frames.map(f => f.cpu / 100)).toFixed(1)}%</span>;
}
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `level` | `"l1" \| "l2" \| "l3"` | Ring buffer level |
| `max` | `number` | Max entries to fetch |

Returns `TtMetrics[]` for L1, `TtAggMetrics[]` for L2/L3.

## useAutoLevel()

Returns the optimal ring buffer level for a given time range.

```ts
import { useAutoLevel } from 'tinytsdk/react';

const level = useAutoLevel('6h');  // → 'l1'
const level = useAutoLevel('3d');  // → 'l2'
const level = useAutoLevel('20d'); // → 'l3'
```

| Range | Returns |
|-------|---------|
| ≤ 1 h | `l1` |
| ≤ 24 h | `l2` |
| > 24 h | `l3` |

## useRawPackets()

Low-level hook. Fires on every binary frame before parsing.
Useful for logging, CSV export, or forwarding to a third-party service.

```ts
import { useRawPackets } from 'tinytsdk/react';

function PacketLogger() {
  useRawPackets((type, payload) => {
    console.log('PKT', type.toString(16), payload.byteLength, 'bytes');
  });
  return null;
}
```

## useTheme()

Returns the current theme tokens.

```ts
import { useTheme } from 'tinytsdk/react';

function Dot() {
  const t = useTheme();
  return <span style={{ color: t.cpu }}>●</span>;
}
```
