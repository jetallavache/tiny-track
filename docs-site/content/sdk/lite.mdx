# tinytsdk-lite

A minimal package containing only the WebSocket client and protocol parser.
No React, no three.js, no components.

## When to use lite vs full

| | `tinytsdk` | `tinytsdk-lite` |
|--|-----------|----------------|
| React components | ✓ | — |
| WebSocket client | ✓ | ✓ |
| Protocol parser | ✓ | ✓ |
| Bundle (gzip) | 2.8 KB core / 23.7 KB React | 2.7 KB |
| Use case | React apps, dashboards | Vanilla JS, Node.js, custom UI |

## Install

```bash
npm install tinytsdk-lite
```

## Usage

```ts
import { TinyTrackClient } from 'tinytsdk-lite';

const client = new TinyTrackClient('ws://your-host:25015');
client.on('metrics', (m) => {
  console.log(`CPU: ${(m.cpu / 100).toFixed(1)}%`);
});
client.connect();
```

## Versioning

Major version is kept in sync with `tinytsdk`. Current: **0.7.x**.

Published to npm via the `sdk-lite/v*` tag trigger in CI.
