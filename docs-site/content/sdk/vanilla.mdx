# Vanilla JS / CDN

No build step required. Include the IIFE bundle from jsDelivr.

## CDN

```html
<script src="https://cdn.jsdelivr.net/npm/tinytsdk/dist/tinytsdk.min.js"></script>
<script>
  const client = new TinyTrack.TinyTrackClient('ws://your-host:25015');

  client.on('metrics', (m) => {
    document.getElementById('cpu').textContent = (m.cpu / 100).toFixed(1) + '%';
    document.getElementById('mem').textContent = (m.mem / 100).toFixed(1) + '%';
  });

  client.connect();
</script>

<span id="cpu">—</span> CPU
<span id="mem">—</span> RAM
```

## Available globals

| Global | Description |
|--------|-------------|
| `TinyTrack.TinyTrackClient` | WebSocket client |
| `TinyTrack.createClient()` | Factory function |
| `TinyTrack.buildCmd()` | Command builder |
| `TinyTrack.parseMetrics()` | Metrics parser |

All protocol constants (`PKT_*`, `CMD_*`, `RING_*`) are also exported.

## Bundle size

`tinytsdk.min.js` — **2.8 KB gzip**. Core only (no React, no three.js).

## TypeScript (ESM)

```ts
import { TinyTrackClient } from 'tinytsdk';

const client = new TinyTrackClient('ws://your-host:25015');
client.on('metrics', (m) => console.log(m));
client.connect();
```
